# LiveTextify: Voice to Text to Insight

**LiveTextify** is a real-time voice-to-text application powered by state-of-the-art neural engines (Whisper, LLaMA, and embedding models). It transforms live audio into structured transcripts, enables AI chat interactions, and provides semantic search capabilities through Retrieval-Augmented Generation (RAG).

## ✨ Features

### 🎙️ Real-Time Transcription
- Low-latency speech-to-text using Whisper models
- Live streaming with visual audio feedback
- Support for multiple languages and translation

### 🧠 AI-Powered Chat
- Context-aware conversations with LLaMA-based LLM
- Session-aware memory and history
- Streaming token generation for real-time responses

### 🔍 RAG & Semantic Search
- Intelligent text chunking and embedding
- Vector similarity search with HNSW indexing
- Context retrieval for enhanced AI responses
- Persistent vector storage with SQLite

### 📱 Cross-Platform Support
- **Desktop:** Windows, macOS, Linux with native frameless window
- **Mobile:** Android (with permissions handling) and iOS support
- Adaptive UI that responds to device form factor

### 🎨 Modern UI/UX
- Clean, dark-themed interface with Material Design
- Clear Mode for distraction-free transcription
- Real-time engine status indicators
- Session management with history
- Customizable settings for all engine parameters
